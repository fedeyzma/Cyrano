# You — Personal Context for the Reply Agent

Copy this file to `fred_context.md` (gitignored) and fill it in with raw context
about yourself. It's injected into the system prompt to ground replies in your
real voice and background.

This is **not** a dating bio, **not** a script, and **not** a list of jokes —
just background so replies feel like you. The agent won't quote it directly.

The more honest and specific this is, the better the replies. Edits take effect
on the next generation (no restart needed).

---

## Core identity
- Name, age, where you're from, where you live.
- Languages you speak (the agent will match the language of the conversation).

## Current life context
- What you do (work/study), day-to-day.

## Personality
- How friends would describe you. Your sense of humor. What you're like.

## Communication style
- How you actually text: short/long, casual/polished, emoji or not, slang.
- Per-language notes if relevant.

## Interests
- The real ones — only surfaced when relevant, never forced.

## Dating / social vibe
- The impression you want to give. What to avoid sounding like.

## Hard rules
- Anything the agent should never do or say as you.
